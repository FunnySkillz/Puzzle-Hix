## Rules

- Do not add paid assets during the 0 EUR phase.
- Do not add assets with unclear redistribution rights.
- Record the URL, license name, and import path for every external asset.
- Prefer original generated assets for brand-critical items: icon, logo, skins, boosters, and store screenshots.
