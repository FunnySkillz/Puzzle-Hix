# PuzzleDungeon

PuzzleDungeon is a planned mobile hybrid-casual puzzle game for Android and iOS. The project is intended to start as a focused MVP with short play sessions, a clear core loop, and a monetization model built around rewarded ads and a one-time "Remove Ads" purchase.

This repository currently contains the Unity project foundation for that work and the project proposal used to define its scope, goals, and delivery approach.

## Current Status

- A Unity project exists and is ready for initial development work.
- The current Unity editor version is `2022.3.15f1`.
- The `Assets` folder does not yet contain meaningful gameplay content.
- No scenes are currently added to build settings.
- The Unity product name is still the temporary default `New Unity Project`.
